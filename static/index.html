Quick Guide

帮助文档

01

第一个课程

在课程页面复制浏览器地址栏中的完整链接，粘贴到主界面输入框。

示例：https://service.icourses.cn/resCourse/web/multi-level?token=你的token
02

第二个课程

处理方式相同：复制带有 token 参数的课程链接，按行粘贴即可。

示例：https://service.icourses.cn/resCourse/web/multi-level?token=你的token
03

第三个课程

先打开课程页面，按 F12 进入控制台，粘贴下面代码获取 token。然后把 token、连字符和课程链接拼成一行。

格式：token-课程链接
示例：你的token-https://higher.smartedu.cn/course/lmc/课程ID
